# AiNovation — Design System

**«Structured AI Architecture» / Структурная ИИ-архитектура.**
B2B-консалтинг по внедрению ИИ для инжиниринговых, проектно-технических и экспертных компаний. Tagline: **NEXT-LEVEL BUSINESS** · «Системный бизнес нового уровня».

> Ключевая формула: **80% строгой B2B-структуры + 20% AI-glow.** AI виден через систему — схемы, карты процессов, узлы, дашборды, аккуратное свечение — а **не** через роботов, мозги или «космос».

---

## Контекст компании / продукт

AiNovation помогает собственникам B2B диагностировать процессы, выбрать приоритетную зону для ИИ и запустить пилотное внедрение без хаоса. Тон — деловой, спокойный, системный, **без хайпа**: партнёр, а не модный AI-стартап. Главный продукт на сегодня — **маркетинговый лендинг** (см. `ui_kits/website/`) с конверсией в форму «мини-диагностики».

## CONTENT FUNDAMENTALS — как написан копирайт

- **Язык:** русский. Обращение на **«вы»**, уважительно, по делу.
- **Тон:** деловой, спокойный, уверенный. Партнёр-эксперт, который умеет управлять внедрением. Не продавец «магии».
- **Casing:** заголовки — обычный регистр (sentence case) либо UPPERCASE в крупных слайд-заголовках. Эйбрау-лейблы — UPPERCASE с трекингом (`letter-spacing: 0.14em`), напр. «СИСТЕМНЫЙ БИЗНЕС НОВОГО УРОВНЯ».
- **Структура мысли:** короткие смысловые блоки, много воздуха, крупные заголовки. B2B-клиент не читает стену текста — он быстро понимает: что вы делаете, для кого, какой первый шаг, почему вам можно доверять.
- **Конкретика вместо общих фраз.** ✅ «Внедряем ИИ туда, где он даёт бизнес-эффект» · «Превращаем хаос в систему». ❌ «Transform Your Workflow», «Нейросети сделают всё за вас».
- **Цифры и результат.** Эффект показываем измеримо: «−40% ручных операций», «6 недель от диагностики до запуска», «×3 скорость подготовки КП».
- **Статусы процесса** — фирменная деталь микрокопирайта: «Карта ИИ-сценариев готова», «Пилот выбран», «Команда обучена», «Контроль качества включён».
- **Эмодзи — нет.** Восклицаний, инфобизнес-тона, крипто/трейдинг-агрессии («95% компаний…») — нет.

## VISUAL FOUNDATIONS — визуальные основы

- **Цвет.** Тёмная база (`#08070B` void → `#0E1020` navy → `#161629` nebula карточки). Текст: frost white / mist / muted. **Один** главный акцент — **AI Violet `#8B5CF6`**; вторичный — **Signal Cyan `#49D6FF`** только как свет AI (линии, узлы, статусы); mint `#47EBA5` — «live». Не делать радугу: акценты — 5–10% площади.
- **Фирменный градиент.** Единственный: `#8B5CF6 → #49D6FF → #D6DEFF`. Только на главной кнопке, hero-визуале, линии процесса и 1–2 декоративных элементах. Не заливать им половину экрана.
- **Типографика.** **Manrope** — бренд и маркетинг (заголовки, текст). **Inter** — UI, дашборды, схемы, числа (tabular-nums). Крупные заголовки, отрицательный трекинг на display (`-0.02em`). Полностью поддерживают кириллицу.
- **Фон.** Сплошные тёмные заливки (не изображения). Глубина — через смену поверхности (void→navy→nebula) и бордеры, а не тени. Свечение — мягкие radial-gradient blobs (violet/cyan), точечно. Без репитов/текстур/паттернов.
- **Карточки.** Фон `#161629`, бордер `1px #2A2A3D`, радиус **20px** (`--radius-lg`), паддинг **32px**. Лёгкий внутренний glow (`--glow-inset`), **без тяжёлых теней**. Glow-вариант добавляет фиолетовую ауру — не более 1–2 на экран. Стеклянный вариант (`glass`) — blur поверх свечения.
- **Радиусы.** Кнопки/инпуты 14–16, карточки 20, крупные панели 28, пилюли — `999px`.
- **Тени / glow-система.** Outer-тени только для плавающих UI-карточек (`--shadow-md/lg`). Свечение — `--glow-accent` (на hover CTA), `--glow-accent-sm`, `--glow-cyan-sm`. Inset hairline `rgba(255,255,255,.05)` для объёма.
- **Бордеры.** Default `#2A2A3D`, hairline `#3A3A52` на приподнятых поверхностях, active = violet.
- **Прозрачность / blur.** Sticky-хедер и стеклянные карточки: `rgba(8,7,11,.72)` + `backdrop-filter: blur(14px)`. Плавающие статус-карточки в hero — `rgba(22,22,41,.85)` + blur.
- **Анимация.** Сдержанная, без баунса. Easing `cubic-bezier(0.22,1,0.36,1)`. Появление карточек (fade+lift), мягкое движение glow, подсветка линий процесса при скролле. Длительности 140 / 220 / 400ms. Уважаем `prefers-reduced-motion`.
- **Hover.** Карточки: lift `−3px` + violet-бордер + лёгкий glow. Кнопка-primary: lift `−1px` + усиленный glow. Ссылки: подчёркивание. **Press:** `scale(0.985)` + `translateY(1px)` — лёгкое сжатие.
- **Иконография изображений.** Холодная, тёмная, технологичная. Никаких роботов, лиц андроидов, лампочек, мозгов, 3D-токенов.
- **Layout.** Контейнер 1240px, поля 80px (desktop) / 22px (mobile), 12 колонок. Между крупными секциями 110–160px. Sticky-хедер — единственный фиксированный элемент.

## ICONOGRAPHY

- **Система:** [Lucide](https://lucide.dev) (CDN, stroke-width **1.75**, line-стиль). У бренда нет собственного набора иконок — Lucide выбран как чистый, тонкий, технологичный line-set, согласующийся с «структура важнее декора». **Это подстановка** (см. CAVEATS).
- **Где:** в карточках услуг/сценариев, в форме, в навигации — нейтральным цветом (`--frost-blue` / `--cyan` для AI-акцента).
- **Брендовый знак (mark):** три соединённые AI-узла с градиентной линией («диагностика → пилот → масштаб») — реализован в `components/core/Logo.jsx` как inline-SVG. Это **единственный** разрешённый hand-built вектор (логотип).
- **Эмодзи:** не используются. Unicode-стрелки (`→`) допустимы в кнопках/ссылках как часть копирайта.
- **Статус-точки:** маленький светящийся кружок в `Badge dot` — сигнал «процесс активен».
